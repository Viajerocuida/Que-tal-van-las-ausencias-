# Que-tal-van-las-ausencias-
Dashboard ejecutivo interactivo de ausentismo para CDE en Viajero Hostels, diseñado para identificar en segundos sedes críticas, diagnósticos frecuentes, focos osteomusculares, psicosociales y crónicos, policonsultantes y alertas tempranas mediante KPIs, filtros dinámicos e insights automáticos.
